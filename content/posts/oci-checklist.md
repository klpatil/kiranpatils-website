Video[https://www.youtube.com/watch?v=Zjkp\_ExgJbA](https://www.youtube.com/watch?v=Zjkp_ExgJbA)NotesSteps to Remember:
1) Edit Applicant Image & Signature as per OCI and keep it handy
2) Fill Online Registration Form ( Part A & Part B)
3) Download the Parental Auth form & Consent Letter from the above link and get notarized and re-upload the parental auth form again in the Registration Form
4) Create a VFS Global account (if you have not already ) to add your minor applicant using ARN and pay your fees
4.a) Include Payment Receipt or money order or cashier check
4.b) send a self-addressed envelope or Money order for a return Envelop (if not paying online)
5) Download Courier Label
6) View the Supportive document Checklist (https://drive.google.com/drive/u/0/folders/1W61jepJKHBly3wE4OiV3Sm3KmadvlQz5)
7) Don't forget to self-attest all the document
8) Use only OCI suggested Fed Ex envelopes and superscribe that is OCI for Minor - By Birth
It is mandatory to upload Applicant's and Applicant's signature images during OCI-Registration.
Both the images must be in jpeg or jpg format, with a max size of 500kb.
The height and width of the Applicant Photo must be equal.
o The minimum dimensions are 200 pixels (width) x 200 pixels (height).
o The maximum dimensions are 1500 pixels (width) x 1500 pixels (height).
The height and width of the Signature Photo must-have aspect ratio 1:3
o The minimum dimensions are 200 pixels (width) x 67 pixels (height).
o The maximum dimensions are 1500 pixels (width) x 500 pixels (height).
Kindly be ready with both these images before starting the registration process.

\- Use Photo Aid App and Select Country as India and OCI Photo. It will cost ~$1.39
\- You can print photos from Walmart for 25 cents.
\- Scan all docs – Both parents’ passport and visa should be in a single PDF \[and PDF Size should be within the limit\]Link[https://services.vfsglobal.com/one-pager/india/united-states-of-america/oci-services/#new-oci](https://services.vfsglobal.com/one-pager/india/united-states-of-america/oci-services/#new-oci)
